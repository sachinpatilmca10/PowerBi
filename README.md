This project is an interactive Power BI dashboard built to analyze Flipkart sales data and uncover insights related to customer behaviour, product performance, and overall business trends. It helps understand key metrics through visually rich and dynamic reports.

Project Overview

The Flipkart Sales Analysis Dashboard provides a complete view of:

Sales Performance (Total Sales, Orders, Profit)

Category & Sub-category Analysis

Sales by State/Region

Customer Demographics & Purchase Trends

Monthly/Quarterly Sales Trends

Top-selling Products and Brands

Low-performing SKUs for business decisions

The dashboard is designed to help stakeholders make informed decisions based on data-driven insights.

Key Features

Fully interactive visuals

Clean and intuitive report layout

Drill-through & filtering enabled

Dynamic slicers for better segmentation

Visual storytelling with KPIs and charts

Tools & Technologies

Power BI Desktop

Data Cleaning & Modeling

DAX Measures for Calculations

Data Visualization Techniques
